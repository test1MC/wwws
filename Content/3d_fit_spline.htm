3D Fit Spline

Create a 3D fit spline through a series of vertices. Creates a curve which is listed in the Parts list under Curves.

3D Fit Spline: Desktop

In the Feature toolbar:

Create a 3D fit spline through a series of vertices or edges that are tangential continuous. Creates a curve which is listed in the Parts list under Curves.

Video example

Video transcript

The 3D fit spline feature tool allows you to create a 3D fit spline through a series of vertices or edges that are tangential continuous. Click the 3D fit spline feature tool on the toolbar. Select the Vertices to form the spline. Check Closed spline if required. Click the green checkmark to accept the new spline. Alternatively, you can create a spline based on edges that are tangential continuous. Click the 3D fit spline feature tool on the toolbar, select Edges, and then select the edges to include in the spline. Click the green checkmark. A new Curve is created in the Curve list that is based on the new spline.
Steps
  1. Click .

  2. Click to select whether to use vertices, or edges at the top of the dialog.
    1. For splines using vertices:
      1. Click to select vertices along which to create the 3D fit spline.
      2. Optionally, check the box to create a closed spline.
      3. If the spline is not closed, you have the option to click a line or a Mate connector (inferred or existing) to select a Start direction.
      4. Enter a value or click and drag the directional arrow in the graphics area to adjust the Start magnitude.
      5. Optionally, click to select Match curvature at start to match the curvature of the edge or face selected as the start direction.
      6. Optionally, click a line or Mate connector (inferred or existing) to select an end direction.
      7. Enter a value or click and drag the directional arrow in the graphics area to adjust the End magnitude.
      8. Optionally, click to select Match curvature at end to match the curvature of the edge or face selected as the end direction.
    2. For splines using edges, click to select tangentially continuous edges with which to create the 3D fit spline.
  3. Click .

A curve is created, listed under Curves in the Parts list. You are not able to show/hide the 3D Fit Spline feature; use the show/hide functionality in the Curves (Parts) list instead.

 3D Fit Spline: iOS

Create a 3D fit spline through a series of vertices, or edges that are tangential continuous. Creates a curve which is listed in the Parts list under Curves.

Steps
  1. Tap the 3D fit spline tool.

  2. Tap to select whether to use vertices, or edges at the top of the dialog.
      1. For splines using vertices:
        1. Tap to select vertices along which to create the 3D fit spline.
        2. Optionally, check the box to create a closed spline.
        3. If the spline is not closed, you have the option to click a line or a Mate connector (inferred or existing) to select a Start direction.
        4. Enter a value or drag the directional arrow in the graphics area to adjust the Start magnitude.
        5. Optionally, select Match curvature at start to match the curvature of the edge or face selected as the start direction.
        6. Optionally, select a line or Mate connector (inferred or existing) to select an end direction.
        7. Enter a value or drag the directional arrow in the graphics area to adjust the End magnitude.
        8. Optionally, select Match curvature at end to match the curvature of the edge or face selected as the end direction.
    1. For splines using edges, select tangentially continuous edges with which to create the 3D fit spline.
  3. Tap the checkmark.

A curve is created, listed under Curves in the Parts list. You are unable to show/hide the 3D Fit Spline feature; use the show/hide functionality in the Curves (Parts) list instead.
